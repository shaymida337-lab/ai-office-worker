## 3. תוכנית הביצוע (3 קבצים, אפס שבירת imports)

**צעד 1 — שכבה A הופכת ל-re-export:** `lib/design-tokens.ts` ייבא מ-`design-system/tokens` ויחשוף את **אותם שמות ייצוא בדיוק** (`colors`, `radius`, `type`...). 59 הצרכנים לא משתנים באות אחת. ערכים שסטו — מקבלים את הקנוני. (`dashboardHome` וסולם ה-px של הדשבורד נשארים כ-legacy מוצהר בתוך הקובץ.)

**צעד 2 — שכבה C מיושרת ידנית:** עדכון 26 המשתנים ב-`globals.css` לערכי B + הערת "synced with design-system/tokens — אל תערוך כאן בלי לעדכן שם". (חיבור runtime דרך `natalieThemeCssVars` נדחה לשלב ה-dark-mode — שינוי התנהגות מיותר עכשיו.)

**צעד 3 — תיקון ה-CTA:** settings:739 → מחלקת `.btn` הסטנדרטית.

**אימות:** build + הסוויטה + הרצת visual-QA (`run-qa.mjs`) לפני/אחרי על הדשבורד וההגדרות.
